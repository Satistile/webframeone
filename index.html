About web:frameone:

web:frameone is a little CSS Framework that have a modern design and is easy to use. The whole Framework is developed by a german developer called Satistile. All properties of this Framework are optimized to work perfectly with a Font called Inter. But more about that topic is in the documentation. The Framework will work with other fonts, but perhaps it don't fit as anticipated. We hope, web:frameone is helpful.

Benefits:

Modern

This Framework has a modern design, which can be used in many ways.

Easy

Open

The entire Framework is open-source and viewable on Git-Hub.
