About web:frameone:

web:frameone is a little CSS Framework that have a modern design and is easy to use. The whole Framework is developed by a german developer called Satistile. All properties of this Framework are optimized to work perfectly with a Font called Inter. But more about that topic is in the documentation. The Framework will work with other fonts, but perhaps it doesn't fit as anticipated. We hope, web:frameone is helpful. We're also on GitHub.

Benefits:

Modern

This Framework has a modern design, which can be used in many ways.

Easy

Every corner of the Framework have a self explaining syntax.

Open

The entire Framework is open-source and viewable on Git-Hub.

How to use:

You can use web:frameone in a few different ways.
When you want to use the newest Version of web:frameone, you must use <link rel="stylesheet" href="https://satistile.github.io/
webframeone/resources/webframeone.css">.
When you want to use a specific full release of web:frameone, you must use <link rel="stylesheet" href="https://satistile.github.io/
webframeone/resources/webframeone-[version].css">.
It's also possible to download the framework from these links.
If you want to use an element which need JavaScript to run (you see this at the language badges at the documentation), you must use <script src="https://satistile.github.io/
webframeone/resources/webframeone.js">.
If you want to use a specific full release of the JavaScript of web:frameone, you must use <script src="https://satistile.github.io/
webframeone/resources/webframeone-[version].js">

1. Introduction:

Welcome in our documentation. Here you can find all official features of web:frameone and sometimes a couple of experimental features. You're welcome to make this framework better and if you don't like anything, you can just fix it and make the framework a bit better. For responsibility, it's useful to use <meta name="viewport" content="width=device-width, initial-scale=1.0"><meta http-equiv="X-UA-Compatible" content="ie=edge"> in your head section.
