About web:frameone:

web:frameone is a little CSS Framework that have a modern design and is easy to use. The whole Framework is developed by a german developer called Satistile. All properties of this Framework are optimized to work perfectly with a Font called Inter. But more about that topic is in the documentation. The Framework will work with other fonts, but perhaps it doesn't fit as anticipated. We hope, web:frameone is helpful.

Benefits:

Modern

This Framework has a modern design, which can be used in many ways.

Easy

Every corner of the Framework have a self explaining syntax.

Open

The entire Framework is open-source and viewable on Git-Hub.

How to use:

You can use web:frameone in a few different ways.
When you want to use the newest Version of web:frameone, you must use <link rel="stylesheet" href="https://satistile.github.io/
webframeone/resources/webframeone.css">.
When you want to use a specific full release of web:frameone, you must use [insert link here].
It's also possible to download the framework from these links.
If you want to use an element which need JavaScript to run (you see this at the language badges at the documentation), you must use <script src="https://satistile.github.io/
webframeone/resources/webframeone.js".
If you want to use a specific full release of the JavaScript of web:frameone, you must use [insert link here]

2. Basic layouts & alignments:

Standard side padding:

CSS

This is the standard side padding. It's responsive and fit's on every device. You can see it on this site at every corner.
</div>
